# SkillBridge - React + TypeScript Implementation

This is a pixel-perfect implementation of an educational administration dashboard based on the provided Figma design.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Lucide React Icons

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Nworah-Gabriel/SkillBridge.git
```

2. Install dependencies:
```bash
   npm install
```
3. Run:

```bash
npm run dev
```
